## ğŸ¯ COME CREARE LE CARTELLE SU GITHUB

GitHub non permette di creare cartelle vuote direttamente. Devi creare un file dentro ogni cartella.

### Procedura per ogni cartella:

1. Nel repository, clicca **"Add file"** â†’ **"Create new file"**

2. Nel campo del nome file, scrivi:
   - Per biennio: `materiali/biennio/.gitkeep`
   - Per terzo: `materiali/terzo/.gitkeep`
   - Per quarto: `materiali/quarto/.gitkeep`
   - Per quinto: `materiali/quinto/.gitkeep`

3. Lascia il contenuto vuoto

4. Clicca **"Commit changes"**

Il file `.gitkeep` Ã¨ una convenzione per mantenere cartelle vuote in Git. Non fa nulla, serve solo a creare la cartella.
